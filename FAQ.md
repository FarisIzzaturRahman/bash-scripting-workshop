## 🙋 Pertanyaan Umum

### Apakah saya perlu pengalaman coding?
**Tidak.** Workshop ini dirancang untuk pemula total. Kalau kamu bisa mengetik dan menyimpan file, kamu sudah cukup siap.

### Saya pakai Windows, apakah bisa ikut?
**Bisa.** Kamu perlu menginstall WSL (Windows Subsystem for Linux). Panduan lengkap ada di [`SETUP.md`](SETUP.md). Sebagai alternatif darurat, ada opsi berbasis browser (lihat di bawah).

### Laptop saya spesifikasinya rendah, apakah masalah?
Tidak masalah. Bash sangat ringan. Yang penting bisa menjalankan terminal dan Zoom bersamaan. Kalau berat, tutup aplikasi lain saat workshop.

### Saya tidak bisa/tidak ingin menginstal apa pun di laptop. Apakah ada solusinya?
**Ya, sangat disarankan menggunakan GitHub Codespaces!** 
Kami telah menyiapkan konfigurasi khusus untuk workshop ini. Cukup klik tombol **Open in GitHub Codespaces** di bagian atas `README.md` pada repositori kami. Semua tools, file latihan, dan ekstensi editor akan terpasang secara instan dan gratis di browser kamu (hanya perlu akun GitHub). Opsi browser lain seperti Google Cloud Shell juga tetap bisa digunakan.

### Apakah materi bisa diakses setelah workshop?
**Ya!** Seluruh repo ini milikmu untuk dipelajari kapan saja. Solusi hands-on, cheatsheet, dan resources tetap tersedia.

### Bagaimana kalau saya tertinggal saat sesi live?
- Jangan panik — semua perintah ada tertulis di modul.
- Ketik `pwd` dan `ls` untuk orientasi ulang (lihat [`TROUBLESHOOTING.md`](TROUBLESHOOTING.md)).
- Tanyakan di chat — instruktur & co-host memantau.
- Sesi direkam (jika diizinkan penyelenggara) — bisa diputar ulang.

### Apakah data biologi yang dipakai asli?
Data sekuens di workshop ini adalah **data sintetis untuk edukasi** (ditandai `educational_only`). Nama gen & organisme nyata digunakan agar terasa realistis, tapi sekuensnya dibuat untuk latihan — aman dan bebas masalah lisensi.

### Setelah bash, sebaiknya belajar apa?
Jalur lanjutan yang disarankan ada di [`resources/README.md`](resources/README.md): Python (BioPython) → R (Bioconductor) → workflow manager (Nextflow/Snakemake).

### Apakah ada sertifikat?
Kebijakan sertifikat ditentukan penyelenggara (OmicsLite). Tanyakan saat pendaftaran atau di sesi penutup.

### Bash, shell, terminal — apa bedanya?
Singkatnya: **terminal** = jendelanya, **shell** = program penerjemah perintah, **bash** = jenis shell paling umum. Penjelasan lengkap ada di [`GLOSSARY.md`](GLOSSARY.md).

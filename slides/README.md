## Yang baru di v2
- **Section "Tanpa Install"** — cara peserta memakai bash di browser tanpa WSL/Ubuntu (rekomendasi: **sandbox.bio**, plus Google Cloud Shell, Replit, GitHub Codespaces, webvm.io).
- **Penjelasan lebih dalam** — konteks "kenapa penting", analogi, contoh diperluas.
- **6 diagram kustom** — termasuk anatomi perintah, pohon sistem file, dan skala kualitas Phred.
- **Callout** — kotak Tip 💡, Peringatan ⚠️, dan Latihan ✍️ (dengan checkbox) di setiap modul.
- **Referensi tambahan** — tabel Error Umum & Solusi, Glosarium, dan jalur belajar lanjutan.
- **Desain** — palet terminal-hijau + aksen amber/cyan, motif "jendela terminal", daftar isi ber-nomor halaman.

## Tanpa install WSL/Ubuntu — ringkas
- **sandbox.bio** ⭐ — terminal bioinformatika di browser, tanpa login/setup, tutorialnya cocok dengan modul ini.
- **Google Cloud Shell** — Linux Debian penuh, 50 jam/minggu gratis (perlu akun Google, usia 18+).
- **Replit / GitHub Codespaces** — IDE browser + terminal (perlu akun).
- **webvm.io** — Linux di browser tanpa login (tool terbatas).

> Untuk mengubah isi: edit langsung di PowerPoint/Word, atau ubah modul sumber di `modules/` lalu regenerasi.
